# 阿康的 Mac 贴贴 v1.0.0

发布于 2026 年 7 月 16 日。

这是品牌升级并聚焦贴窗体验后的首个正式版本。

## 更新内容

- 产品统一命名为“阿康的 Mac 贴贴”。
- 移除未完成的截图与资讯入口，主面板只保留贴窗功能。
- 支持 15 种窗口布局和 `⌃⌥` 全局快捷键。
- 安装包同时支持 Apple 芯片与 Intel Mac（Universal 2）。
- 修复正常启动与终端启动生命周期不一致的问题。
- 防止菜单栏进程被 macOS 自动终止。
- 快捷键注册增加真实结果记录与有限重试。
- 优化多显示器连续按键跳屏逻辑。
- 首次安装只主动请求一次辅助功能权限，后续不重复弹窗。
- 设置页新增辅助功能权限状态及系统设置入口。
- 使用固定 Bundle ID 与 designated requirement 进行本地签名，减少升级后授权失效。
- 关于页面以非捆绑方式推荐“阿康的 AI 语音输入法”。

## 发布附件

`AkangMacTieTie-v1.0.0-macos.zip`

## 签名说明

当前附件为 ad-hoc 本地签名版本，尚未使用 Apple Developer ID 签名和公证。首次打开时可能需要右键选择“打开”，或在“系统设置 → 隐私与安全性”中允许运行。
