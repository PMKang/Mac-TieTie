## 一、产品概述

### 1.1 背景

macOS 缺乏原生窗口吸附功能（Windows 的 Win+方向键），第三方工具 Magnet 售价 $4.99，有 13.4 万用户付费。
Mac贴贴 以「完全免费 + 开源」切入，面向中文 Mac 用户，工具本身引流关注公众号「阿康AI探索号」。

### 1.2 目标用户

- 主力：每天使用 Mac 的效率党（产品、设计、开发、运营）
- 特征：习惯 Windows 多窗口操作，换 Mac 后不适应
- 渠道：公众号「阿康AI探索号」粉丝 + 小红书/知乎自然传播

### 1.3 核心价值

| 用户痛点 | Mac贴贴 方案 |
|---------|------------|
| macOS 无窗口吸附 | 16种吸附位置 + 全局热键 |
| Magnet 要花钱 | 全部功能永久免费 |
| 没有中文工具支持 | 中文界面 + 中文社区 |

## 三、用户流程

### 3.1 首次使用流程

```
下载 Mac贴贴.zip → 解压 → 将 Mac贴贴.app 拖入 /Applications
    ↓
右键点击 App → 选择「打开」→ 弹窗中点「打开」（绕过未签名提示）
    ↓
菜单栏出现 ⊞ 图标
    ↓
系统提示授权辅助功能 → 点「打开系统设置」→ 勾选 Mac贴贴
    ↓
切换到任意窗口，按快捷键即可吸附
```

### 3.2 日常使用流程

```
按下 ⌃⌥ + 方向键/字母键
    ↓
HotkeyManager 接收 Carbon 事件
    ↓
WindowManager 用 AXUIElement 获取前台窗口
    ↓
计算目标 frame（基于当前屏幕 visibleFrame）
    ↓
AXUIElement 设置窗口 position + size
```

### 3.3 权限说明

- **辅助功能（Accessibility）**：必须授权，用于通过 AXUIElement API 控制其他 App 窗口位置和大小
- 每次 Xcode 重新编译会使权限失效，需重新授权（发布版无此问题）

## 五、技术架构

### 5.1 整体架构

```
Mac贴贴 App（Swift + AppKit + SwiftUI）
    ├── Carbon API          → 全局热键监听（不抢焦点）
    ├── AXUIElement API     → 窗口位置/大小控制（需 Accessibility 权限）
    └── ServiceManagement   → 登录时自动启动
```

无服务端依赖，完全本地运行。

### 5.2 Swift App 文件结构

```
MacPastie/
├── MacPastieApp.swift           # App 入口，NSApplicationDelegateAdaptor
├── AppDelegate.swift            # 菜单栏图标、Popover、偏好/关于窗口管理
├── Core/
│   ├── WindowManager.swift      # AXUIElement 窗口控制，visibleFrame 计算
│   └── HotkeyManager.swift      # Carbon RegisterEventHotKey，16个热键
└── Views/
    ├── MenuPanelView.swift       # Tab 面板主容器 + 二维码引流条
    ├── SnapGridView.swift        # 3x3 可视化吸附网格
    ├── PreferencesView.swift     # 偏好设置（HotkeyPrefsView + GeneralPrefsView）
    └── AboutView.swift           # 关于窗口（版本 + 更新日志 + 二维码）
```

### 5.3 关键技术点

| 技术点 | 实现方式 | 注意事项 |
|--------|---------|---------|
| 全局热键 | Carbon `RegisterEventHotKey` + `InstallEventHandler` | 不依赖 App 焦点，任何前台 App 均可触发 |
| 窗口控制 | `AXUIElement` kAXPositionAttribute / kAXSizeAttribute | 必须 `AXIsProcessTrusted()` 为 true |
| 菜单栏常驻 | `NSApp.setActivationPolicy(.accessory)` | 不出现在 Dock |
| 弹出面板 | `NSPopover` + `NSHostingController<MenuPanelView>` | `.transient` 行为，点外自动关闭 |
| 设置/关于窗口 | `AppDelegate.shared` 静态引用 + 手动 `NSWindow` 管理 | 不用 SwiftUI Settings scene（有冲突） |
| 登录启动 | `SMAppService.mainApp.register()` | macOS 13+ |

## 七、发布方式

### 7.1 分发

- GitHub 开源（MIT 协议）：`https://github.com/PMKang/Mac-TieTie`
- GitHub Releases 发布 `.zip`（含已编译 `.app`）
- 首发渠道：公众号「阿康AI探索号」+ 小红书

### 7.2 安装说明

1. 下载 `Mac贴贴.zip` → 解压 → 拖入 `/Applications`
2. 右键 → 打开 → 弹窗点「打开」（绕过未签名提示）
3. 如仍无法打开：`xattr -cr /Applications/Mac贴贴.app`
4. 首次使用授权辅助功能权限

### 7.3 源码编译

```bash
brew install xcodegen
git clone https://github.com/PMKang/Mac-TieTie.git
cd Mac-TieTie/MacPastie
xcodegen generate
open MacPastie.xcodeproj
```
